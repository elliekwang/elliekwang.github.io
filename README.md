## Projects

**League of Legends - Neeko Rework**
- This project was for my DSC 80 class (Practice and Application of Data Science) at UCSD.
- My partner and I worked with datasets from the game League of Legends to answer multiple questions we had, including running permutation tests, created and trained binary classification models, feature engineered columns using scikit-learn, and performed fairness analysis. 
- To learn more: [GitHub repository](https://github.com/elliekwang/neeko_rework/blob/main/neeko_rework.ipynb) contains our findings, code, visualizations, etc.
